# Nodoff — Chrome Web Store Description

## Short Description (132 chars max)

Sleep timer for browser tabs — smoothly fades out and pauses audio/video after your chosen timeout. No data collected, ever.

## Detailed Description

**Fall asleep to music, podcasts, or videos without worrying about them playing all night.**

Nodoff is a simple sleep timer for Chrome. Set a timer on any tab, and when it expires, Nodoff smoothly fades the volume to zero over 10 seconds before pausing all audio and video on the page.

**Key Features:**

- **Quick Presets** — Start a 15, 30, 45, or 60-minute timer with one click
- **Custom Duration** — Set any timer length you need
- **Smooth Fade-Out** — Volume gradually decreases over 10 seconds before pausing, so you're never jolted awake
- **Per-Tab Timers** — Each tab gets its own independent timer — set different timers for different tabs
- **Lightweight** — No background resource usage when no timers are active

**Privacy First:**

Nodoff collects zero data. No analytics, no tracking, no external servers. All timer state is session-scoped and clears automatically when you close your browser. Your browsing activity stays entirely on your device.
