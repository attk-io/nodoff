# Phase 8: Chrome Web Store Submission

**Goal:** Published extension.

## Steps

1. Create Chrome Web Store developer account ($5 one-time fee) if needed
2. Upload `.zip` via [Developer Dashboard](https://chrome.google.com/webstore/devconsole)
3. Fill listing: description, category ("Productivity"), screenshots, icons
4. Declare permissions justifications (`storage`, `alarms`, `activeTab`, `scripting`)
5. Set visibility (public or unlisted for initial testing)
6. Submit for review

## Verification

Extension appears in Chrome Web Store (1-3 business days). Install from store on clean profile, verify functionality.
